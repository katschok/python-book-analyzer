# python-book-analyzer

Java lerne ich im Studium, aber dieses Python-Programm habe ich privat zu Hause geschrieben. Es analysiert Textdateien (z. B. ganze Bücher), zählt die Wörter und zeigt an, welche Buchstaben wie oft vorkommen.

So geht's:

Programm starten: Einfach im Terminal die main.py aufrufen und den Pfad zum Buch (der .txt Datei) angeben.

Ergebnis: Das Programm zeigt die Wortanzahl und eine Liste der Buchstaben an, sortiert nach Häufigkeit.

Befehl für das Terminal:

python3 main.py books/frankenstein.txt
