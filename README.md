# python-book-analyzer

Java lerne ich im Studium, aber dieses Python-Programm habe ich privat zu Hause geschrieben. Es analysiert Textdateien, zählt die Wörter und zeigt an, welche Buchstaben wie oft vorkommen.

So geht's:

Programm starten: Öffne dein Terminal in diesem Ordner.

Befehl eingeben: Ich habe die Datei frankenstein.txt schon hochgeladen, damit man das Programm direkt testen kann. Gib einfach das hier ein:
```
python3 main.py frankenstein.txt
```


Ergebnis:
Das Programm zeigt dir dann sofort an, wie viele Wörter das Buch hat und wie oft jeder Buchstabe vorkommt (sortiert von häufig nach selten).
